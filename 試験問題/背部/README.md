# 背部範囲 問題解説 一覧

脊柱の靭帯、椎骨の構造、固有背筋、背部の筋の分類など、講義資料「背部」に対応した問題の解説です。

## 頻出項目・覚えるべき内容

| 項目 | 暗記すべき内容 |
|------|----------------|
| **椎骨・腹側の順** | 前縦靭帯＞椎体＞後縦靭帯＞椎弓＞棘突起 |
| **線維軟骨結合** | 恥骨結合・椎間円板。歯と歯槽＝釘植 |
| **靭帯の付着** | 前十字靭帯＝脛骨前顆間区。内側側副靭帯＝脛骨（腓骨ではない） |
| **背部の筋** | 浅層＝僧帽筋・広背筋・菱形筋・肩甲挙筋。深層＝脊柱起立筋・横突棘筋 |
| **固有背筋** | 腸肋筋・最長筋・棘筋、横突棘筋（半棘筋・多裂筋・回旋筋） |
| **関節** | 蝶番関節＝指節間関節 |
| **運動の定義** | 屈曲・伸展・外転・内転・外旋・内旋と股関節の筋（記述で出題） |

※序論・上肢・頸部と重複出題される。
