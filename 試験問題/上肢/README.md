# 上肢範囲 問題解説 一覧

講義資料（上肢１〜５）の範囲に対応した、各年度試験の上肢関連問題の解説です。

## 頻出項目・覚えるべき内容

| 項目 | 暗記すべき内容 |
|------|----------------|
| **腕神経叢** | C5〜T1。幹：上(C5-C6)・中(C7)・下(C8-T1)。筋皮(外側)・腋窩(後)・橈骨(後)・正中(外+内)・尺骨(内)。分娩麻痺＝上神経幹損傷→前腕尺側の感覚障害は該当しない |
| **手根管症候群** | 正中神経→母指〜環指橈側の掌側の感覚障害 |
| **三角筋・小円筋** | ともに腋窩神経 |
| **肘関節動脈網** | 上腕動脈の側副路。腋窩動脈閉塞時の側副路 |
| **橈骨動脈** | 後骨間動脈は**尺骨動脈**から（橈骨動脈からではない） |
| **虫様筋** | 深指屈筋腱→指背腱膜。MP屈曲・IP伸展。1-2＝正中、3-4＝尺骨 |
| **上腕区画** | 前＝二頭筋・上腕筋・烏口腕筋(筋皮)。後＝三頭筋・肘筋(橈骨) |
| **動脈触知** | 上腕動脈＝二頭筋腱の内側。足背動脈＝長母指伸筋腱と長指伸筋腱の間 |

※頸部・感覚器・頭部と同一問題が重複出題される。
