# 神経解剖学 範囲別解説

講義資料（**神経解剖学**）の範囲に対応する試験問題の解説です。感覚器・頭部・頸部・胸部・腹部の各範囲から神経解剖学該当問題を抽出し、**問題文・選択肢（すべて）・正解・解説**の順にまとめています。

## 頻出項目・覚えるべき内容

| 項目 | 暗記すべき内容 |
|------|----------------|
| **頭蓋の孔** | 上眼窩裂＝動眼・滑車・外転・眼神経。卵円孔＝下顎神経。正円孔＝上顎神経。茎乳突孔＝顔面神経。内耳道＝顔面・内耳神経 |
| **視覚伝導路** | 網膜→視神経→視交叉→視索→外側膝状体→視放線→一次視覚野。視交叉正中＝両耳側半盲。左視索＝右同名半盲 |
| **錐体路** | 内包→大脳脚→橋→延髄錐体→錐体交叉。**内側毛帯**は体性感覚で錐体路に非関与 |
| **小脳** | 前庭小脳・脊髄小脳・橋小脳。**歯状核**＝橋小脳の出力核（前庭小脳ではない） |
| **後索・前外側路** | 後索＝同側、前外側路＝対側 |
| **副交感神経** | 節前＝動眼・顔面・舌咽・迷走（三叉は含まない） |
| **脳の動脈** | 網膜＝眼動脈（内頸動脈）。顎動脈の分布に網膜は含まれない |
| **室間孔** | 側脳室と第三脳室を連絡 |
| **視交叉上核** | 概日リズム |

※感覚器・頭部・頸部・上肢・胸部から該当問題が抽出される。

---

## 講義資料との対応

| 講義資料（神経解剖学フォルダ内） | 主な内容 |
|--------------------------------|----------|
| 脊髄、伝導路 | 脊髄の構造・断面、後索・前外側路、錐体路、脊髄の動脈 |
| 脳幹、間脳 | 脳神経核、菱形窩、視床の中継核 |
| 小脳、大脳 | 小脳の機能的区分、大脳皮質、基底核、辺縁系 |
| 視覚・聴覚・平衡覚・嗅覚・味覚 | 各伝導路、視交叉と視野欠損 |
| 脳血管髄膜脳室 | 脳を栄養する動脈、髄膜、脳室・脳脊髄液、血液-脳関門 |
