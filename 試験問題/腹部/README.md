# 腹部範囲 問題解説 一覧

胃、十二指腸、肝臓・門脈、膵臓、大腸、鼠径管、泌尿器系、子宮、腹腔・骨盤腔の自律神経など、講義資料「腹部」に対応した問題の解説です。

## 頻出項目・覚えるべき内容

| 項目 | 暗記すべき内容 |
|------|----------------|
| **胃** | 胃底＝**上部**の膨らみ（下部ではない）。筋層＝縦走筋は外層 |
| **門脈・肝門** | 肝門＝門脈・肝動脈・胆管（肝静脈は通らない） |
| **膵臓** | 腹膜後器官。脾動脈＝膵体部の**上縁**（前面ではない） |
| **下行結腸** | **下腸間膜動脈**で栄養（上腸間膜動脈ではない） |
| **腎臓** | 左卵巣静脈＝左腎静脈に合流。左腎臓は十二指腸に接しない |
| **小腸** | 絨毛の高さ0.5〜1.2mm |
| **精巣腫瘍** | 初発が肉眼的血尿とならない |
| **中殿筋** | 不全→片脚立位時に骨盤が傾く |
| **鼠径管** | 精索・子宮円索・腸骨鼠径神経（陰部枝は通らない） |

※胸部・序論・下肢・神経解剖学・感覚器と重複出題される。
