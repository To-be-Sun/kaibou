# 序論範囲 問題解説 一覧

解剖学的正位、断面・方向用語、解剖の種類、骨盤、正中の構造など、講義資料「序論」に対応した問題の解説です。

## 頻出項目・覚えるべき内容

| 項目 | 暗記すべき内容 |
|------|----------------|
| **解剖学的正位** | 立位、上肢は**下垂**、手掌は**前**（内側ではない） |
| **臓器の位置** | 最も右＝肝臓。最も左＝脾臓。正中にない＝視放線 |
| **体表触知** | 触知できない＝翼状突起 |
| **骨盤** | 産科的真結合線＝最も小さい。女性＝恥骨下角が大きい |
| **靭帯** | 烏口鎖骨靭帯＝菱形＋円錐。前十字靭帯＝脛骨前顆間区 |
| **解剖の種類** | 教育＝系統解剖 |
| **胸骨角** | 第2肋骨、T4-5の高さ |
| **基本用語** | 起始・停止、終動脈、側副路、門脈 |

※腹部（臓器・骨盤）、背部（靭帯）、上肢・頸部と重複出題される。
