# 感覚器範囲 問題解説 一覧

眼（外眼筋、眼房水、対光反射、視覚伝導路）、耳（鼓室、耳小骨、聴覚）、味覚など、講義資料「感覚器」に対応した問題の解説です。

## 頻出項目・覚えるべき内容

| 項目 | 暗記すべき内容 |
|------|----------------|
| **毛様体筋** | 遠近調節（瞳孔括約筋・散大筋ではない） |
| **眼房水** | 毛様体で産生、Schlemm管から流出。眼圧10〜21 mmHg |
| **対光反射** | 視神経→動眼神経。両眼縮瞳 |
| **上眼窩裂** | 動眼・滑車・外転・眼神経。散瞳・眼球運動障害の責任部位 |
| **外眼筋** | 上直筋＝上内側注視。上斜筋＝下転・外転・内旋。下斜筋＝上外側注視 |
| **鼓室** | 通過する主な神経＝顔面神経（舌咽神経の枝は誤り）。耳小骨＝ツチ・キヌタ・アブミ骨 |
| **味覚** | 舌前2/3＝鼓索神経。舌後1/3＝舌咽神経 |
| **聴覚** | 中継核＝内側膝状体 |
| **視覚伝導路** | 網膜→視神経→視交叉→視索→外側膝状体→視放線→一次視覚野。視交叉正中障害＝両耳側半盲 |
| **内耳道** | 顔面・内耳神経（咀嚼筋＝三叉神経で通らない） |

※上肢（手根管・虫様筋・三角筋・小円筋・橈骨動脈）、頭部・神経解剖学と重複出題される。
