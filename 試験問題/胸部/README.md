# 胸部範囲 問題解説 一覧

胸郭・胸壁の筋、縦隔、心臓・心膜、肺、気管・気管支、食道、奇静脈、交感神経、胸管など、講義資料「胸部」に対応した問題の解説です。

## 頻出項目・覚えるべき内容

| 項目 | 暗記すべき内容 |
|------|----------------|
| **横隔膜の裂孔** | 食道裂孔＝食道・迷走神経（**胸管は通らない**）。大動脈裂孔＝大動脈・胸管。腔静脈孔＝下大静脈 |
| **心臓** | 洞房結節＝上大静脈開口部付近。卵円窩＝右心房の房間中隔。冠状溝＝心房と心室の境。心尖＝第5肋間隙 |
| **肋間** | 肋間神経・動静脈＝**内肋間筋と最内肋間筋の間** |
| **縦隔** | 中縦隔＝心臓 |
| **気管** | 馬蹄型（U字型）軟骨、後方は膜性壁 |
| **細気管支** | 軟骨で覆われない |
| **心膜の痛み** | 横隔神経（C3-C5） |
| **交感神経** | 節前＝白交通枝、節後＝灰白交通枝 |

※腹部・頸部・頭部・神経解剖学と重複出題される。
