# 頭部範囲 問題解説 一覧

脳神経、頭蓋の孔、顔面神経・三叉神経、翼口蓋窩、鼓室、味覚・聴覚・視覚伝導路など、講義資料「頭部」に対応した問題の解説です。

## 頻出項目・覚えるべき内容

| 項目 | 暗記すべき内容 |
|------|----------------|
| **頭蓋の孔** | 上眼窩裂＝動眼・滑車・外転・眼神経。正円孔＝上顎神経。卵円孔＝下顎神経。茎乳突孔＝顔面神経。内耳道＝顔面・内耳神経 |
| **対光反射** | 求心路＝視神経、遠心路＝動眼神経。両眼縮瞳 |
| **視覚伝導路** | 網膜→視神経→視交叉→視索→外側膝状体→視放線→一次視覚野。視交叉正中障害＝両耳側半盲 |
| **味覚** | 舌前2/3＝鼓索神経。舌後1/3＝舌咽神経 |
| **鼓室** | 通過する主な神経＝顔面神経（舌咽神経の枝は誤り） |
| **外眼筋** | 上直筋＝上内側注視。上斜筋＝下転・外転・内旋（滑車神経） |
| **舌下神経障害** | 舌の萎縮・患側への偏位 |
| **顎舌骨筋** | 三叉神経（顔面神経ではない） |

※感覚器（鼓室・外眼筋・視覚伝導路・対光反射・味覚）、頸部・上肢と重複出題される。
