# 頸部範囲 問題解説 一覧

甲状腺、頸部の筋、外頚動脈など、講義資料「頸部」に対応した問題の解説です。

## 頻出項目・覚えるべき内容

| 項目 | 暗記すべき内容 |
|------|----------------|
| **腕神経叢** | C5〜T1、根・幹・束、筋皮・腋窩・橈骨・正中・尺骨神経の分岐と支配域 |
| **肩甲吻合** | 肩甲上動脈・肩甲背動脈・肩甲回旋動脈（肩甲骨背面で吻合） |
| **筋皮神経・烏口腕筋** | 烏口腕筋を貫く＝筋皮神経。主作用＝肩関節の屈曲 |
| **手根管症候群** | 正中神経→母指〜環指橈側の感覚障害 |
| **烏口鎖骨靭帯** | 菱形靭帯＋円錐靭帯（肩鎖靭帯ではない） |
| **上腕骨幹部骨折** | 橈骨神経損傷→下垂手 |
| **甲状腺** | 輪状軟骨〜気管上部の腹側。上甲状腺動脈＝外頚動脈、下＝鎖骨下動脈 |
| **外頚動脈** | 顎動脈・顔面動脈・舌動脈・上甲状腺動脈など |

※上肢・序論・胸部・感覚器・頭部と同一問題が重複出題される。
