# 下肢・骨盤範囲 問題解説 一覧

講義資料に「下肢」「骨盤」の独立した範囲がないため、これまで他範囲でカバーされていなかった問題をまとめています。股関節・膝関節・下腿の筋・骨盤支持・腓骨神経など、下肢・骨盤に関連する問題の解説です。

## 頻出項目・覚えるべき内容

| 項目 | 暗記すべき内容 |
|------|----------------|
| **中殿筋** | 片脚立位時の骨盤支持に最重要。不全→反対側に骨盤が傾く歩行（トレンデレンブルグ） |
| **下腿の区画** | 前区画＝足背屈（前脛骨筋など）。腫脹で足背屈困難・足背動脈拍動減弱 |
| **距骨下関節** | 足の内反・外反に最も関与 |
| **股関節** | 球関節。中殿筋＝外転筋 |
| **腓骨神経** | 足背屈・外反、足背の感覚。麻痺→足背屈不能 |
| **筋裂孔** | 腸腰筋の下を大腿神経・血管が通過（腹部範囲と重複） |

※序論（骨盤計測）、腹部（鼠径管・泌尿器）、背部（靭帯）と重複出題される。
